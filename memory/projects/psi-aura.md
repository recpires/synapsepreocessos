# Psi Aura

**Tipo:** SaaS
**Setor:** Psicologia / Saúde mental
**Status:** Ativo

## Descrição
Software de gestão para psicólogos. Possíveis funcionalidades: agenda de consultas, prontuários digitais, gestão de pacientes, cobrança.

## Notas
Nome sugere posicionamento premium/wellness. Setor de saúde exige atenção à LGPD (dados sensíveis de pacientes).
