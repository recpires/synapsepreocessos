# CRM Nexio

**Tipo:** SaaS / Produto próprio
**Setor:** Gestão de relacionamento com clientes
**Status:** Ativo

## Descrição
CRM desenvolvido pela Synapse Code. Pode ser usado internamente e/ou vendido como produto para clientes.

## Notas
Produto estratégico — CRM próprio posiciona a Synapse Code como referência em gestão. Potencial de uso interno para gerenciar leads e clientes da própria empresa.
